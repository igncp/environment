# Messaging

## Slack

- Settings > Notifications > Notify me about: Nothing. This allows to periodically check for notifications but not to be distracted by them while working.
- Settings > Notifications > Use different settings for mobile devices: Direct messages, mentions & keywords. This is useful to check for emergencies in mobile devices.
- Update theme:
    - https://github.com/widget-/slack-black-theme
    - https://github.com/widget-/slack-black-theme/issues/48#issuecomment-382017521

# Skype

- Windows:
    - Use App from Store
    - Update theme to dark
- Disable notifications. This allows to periodically check for notifications but not to be distracted by them while working (not for phone).
