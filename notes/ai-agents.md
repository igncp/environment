# AI Agents

## GitHub Copilot CLI 快捷鍵

### 全局

- **Ctrl+T** - 切換顯示模型推理過程（隱藏 "The user wants me to..." 等訊息）
- **Ctrl+P** - 運行命令時保留輸入
- **Shift+Tab** - 切換模式（interactive → plan）
- **Ctrl+O** - 展開最近嘅時間線（當無輸入時）
- **Ctrl+E** - 展開所有時間線（當無輸入時）
- **↑ / ↓** - 瀏覽命令歷史
- **!** - 喺本地 shell 執行命令（繞過 Copilot）
- **Esc** - 取消當前操作
- **Ctrl+C** - 取消操作 / 清除輸入 / 退出
- **Ctrl+D** - 關閉
- **Ctrl+L** - 清除屏幕

### 編輯

- **Ctrl+A** - 移動到行首（輸入時）
- **Ctrl+E** - 移動到行尾（輸入時）
- **Ctrl+H** - 刪除前一個字符
- **Ctrl+W** - 刪除前一個單詞
- **Ctrl+U** - 從游標刪除到行首
- **Ctrl+K** - 從游標刪除到行尾
- **Meta+←/→** - 按單詞移動游標

### 常用命令

- **/model** - 選擇 AI 模型
- **/diff** - 檢視當前目錄嘅更改
- **/review** - 運行代碼檢視代理分析更改
- **/help** - 顯示交互命令幫助
- **/theme** - 檢視或配置終端主題
- **/streamer-mode** - 切換串流模式（隱藏預覽模型名稱同配額詳情）
